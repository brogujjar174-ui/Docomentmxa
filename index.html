  • Character Modeling: Blender & Unity

  • Map Creation: Unity Terrain / ProBuilder & Blender (on Windows PC)

5. Map & Game Modes

  • Map Name: Saru
  • Map Size: 500m x 500m (Squad Mode Supported)
  • Game Mode: Battle Royale (Survival Style)
  • In-Game Items / Loot: Guns, Ammo, Health Packs/Medkits, Vests, and Helmets.
  • Environmental System: Shrinking Safe Zone (Blue Zone).

6. Customization & Cosmetics (Inventory System)

  • Head: 1 Slot (Hats, Caps, Hair Styles)
  • Mask: 1 Slot (Face Masks, Glasses)
  • T-Shirt: 1 Slot (Upper Body Clothes)
  • Bottom: 1 Slot (Pants, Shorts)
  • Gun Skins: Custom cosmetics for Desert Eagle, UMP, AK, and AWM.

7. Server & Database Infrastructure

  • Local Server: Offline environment for testing physics & mechanics.
  • Main Server Database: Main cloud database for permanent player profiles.

Player Account Data Structure

Data Field Data Type Description
Name String Player's in-game nickname

Character ID / String Currently equipped character (Mie, Hetro, or Simpa)
Gun Skins Array / List Owned weapon cosmetics inventory
Equipments Array / List Unlocked Head, Mask, T-Shirt, and Bottom items
Gold Integer Free in-game currency earned from playing
Diamond Integer Premium currency via store purchases
